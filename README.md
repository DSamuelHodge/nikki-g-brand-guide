## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/DSamuelHodge/nikki-g-brand-guide.git
   cd nikki-g-brand-guide
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to view the brand guide.

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
